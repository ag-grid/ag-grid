Introducing Angular Grid

Angular Grid is the first high performance and feature rich data grid for AngularJS.

Light weight, minimal footprint, no external dependencies.

Hybrid AngularJS & DOM Engine

The Angular Grid engine uses AngularJS where required and native Javascript at all other times, giving the best performance. A grid written solely in AngularJS will be inherently slower.
